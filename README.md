# VOTL
 Voice of the Lord - discord bot written on Java using JDA library with Chewtils extention

## Config file
 config.json:
 ```json
 {
 	"bot-token": "",
	"owner": "name#0000",
	"bot-invite": "generated link for inviting the bot"
 }
 ```
